vim-goshgolly
==========

Grayscale and neon-pastel 16-color palette and syntax highlighting for vim.

Requirements
------------

- [Plug](https://github.com/junegunn/vim-plug)

Installation
------------

  1. Add `Plug 'kroogs/vim-goshgolly'` to your .vimrc
  2. Run `:PlugInstall`
  3. Run `:colo goshgolly`
